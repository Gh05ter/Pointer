## ZoomEye And Seebug App Tools

Pointer旨在通过ZoomEye 的强大的网络空间搜索能力以及SeeBug漏洞平台了解第一手的漏洞信息打造一款随时随地网络漏洞探索工具。期望后面APP可以集成Pocsuite,那么三剑合一，将会在江湖引起大的风波。

现在Pointer支持一下功能：

1. 可以查看一手的漏洞信息
2. 支持ZoomEye 登录功能
3. ZoomEye强大的Web Search
4. ZoomEye叼叼的Host Search
5. 通过获取的URL可以进行访问。
